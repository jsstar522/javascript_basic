# Buyer Profile Project

주소록 관리 앱입니다. 연락처를 `추가/삭제/수정`하고 `검색`이 가능하며, `즐겨찾기`로 추가할 수 있습니다. 모든 연락처의 정보는 `contacts` state 배열에 저장되며 이 **배열을 어떻게 렌더링하느냐에 따라 다른 화면에서 다른 내용을 보여줍니다.**
